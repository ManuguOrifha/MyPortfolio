Restaurant Menu — Kota Corner

My first real web project — a fully interactive restaurant menu page built for a South African street-food concept centred around the beloved kota. The site lets users browse a curated menu of loaded quarter-loaf sandwiches, view item descriptions, and experience a clean, food-forward design that makes your mouth water before you even order.

Built with pure HTML, CSS, and JavaScript — no frameworks, no shortcuts. Every layout decision, hover effect, and menu card was crafted by hand. This project taught me how to think about user experience, visual hierarchy, and building for a real audience rather than just a class assignment.

A sleek, fully functional browser-based calculator that handles real arithmetic — addition, subtraction, multiplication, division, and chained operations — all without a single library or framework. Just clean, well-structured JavaScript doing exactly what it's supposed to do.

What made this project valuable wasn't the maths — it was the logic. Managing operator precedence, handling edge cases like division by zero, and keeping the display accurate taught me how to think like an engineer: systematically, precisely, and with the user always in mind.
